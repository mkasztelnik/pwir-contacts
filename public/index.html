Simple REST API for managing contacts

  • List contacts
  • Show contact
  • Create new contact
  • Update contact
  • Destroy contact
URL /api/v1/contacts
Method GET
URL params None
Data params None
Success response

Code 200

Example 

{
  "contacts": [
    {
      "id": 1,
      "first_name": "John",
      "last_name": "Doe",
      "email": "john@doe.pl",
      "telephone": "123123123",
      "address": "Some address"
    }, {
      "id": 2,
      "first_name": "All",
      "last_name": "Bundy",
      "email": "all@bundy.pl",
      "telephone": "321321321",
      "address": "Another"
    }
  ]
}
Error responses None
Sample call

Bash (with curl) 

curl -v https://pwir-contacts.herokuapp.com/api/v1/contacts
URL /api/v1/contacts/:id
Method GET
URL params id - Contact id
Data params None
Success response

Code 200

Example 

{
  "contact": [
    {
      "id": 1,
      "first_name": "John",
      "last_name": "Doe",
      "email": "john@doe.pl",
      "telephone": "123123123",
      "address": "Some address"
    }
  ]
}
Error responses

Error response 404

Example 

{
  "message":"Record not found",
  "type":"general"
}
Sample call

Bash (with curl) 

curl -v https://pwir-contacts.herokuapp.com/api/v1/contacts/1
